Esta pasta contem recursos para uso com o mtcli, quais sejam: atalhos de comandos, marcadores, configuracoes de ativos e ferramentas.

Atalhos
g range do dia
rm - range médio de 14 períodos do diário
ma média móvel de 20 períodos do diário
n - volume de negócios do diário

Marcadores
a preço de ajuste
lta - linha de tendência de alta
lca - linha de canal de alta
ltb - linha de tendência de baixa
lcb - linha  de canal de baixa



Ferramentas
trx exibe as 3 regiões de uma lateralidade
pba - Exibe as 3 regiões de recuo de uma pernada de alta
pbb - exibe as 3 regiões de recuo de uma pernada de baixa
em - calcula expectativa matemática de um trade system
ro - calcula o range operacional a partir de um range médio do diário
c - calcula os preços de stop gain e stop loss   de uma compra
v - calcula os preços de stop gain e stop loss   de uma venda
ls - exibe o próximo preço de uma linha superior ligando os topos
li - exibe o próximo preço de uma linha inferior ligando os fundo



instale
pip install kt-line

pip install kt-osc
